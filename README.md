# MEMORY-GAME
Advanced Memory Game using Python Tkinter

🧠 Memory Game Ultra

A modern and interactive Memory Card Game built using Python and Tkinter.
The game challenges players to match pairs of cards with smooth animations, real-time tracking, and an attractive user interface.

🚀 Features
✨ Beautiful modern GUI (dark theme)
🎴 Smooth card flip animation
⏱️ Real-time timer
🎯 Move counter system
🏆 Leaderboard (Top 5 scores saved)
🎮 Multiple difficulty levels (Easy, Medium, Hard)
💾 Data storage using JSON
🖱️ Hover effects and responsive UI
🎉 Winning popup screen

🛠️ Technologies Used
Python
Tkinter (GUI)
JSON (Data Storage)
Random Module
Time Module

▶️ How to Run
Run using Python:
python game.py

Run using EXE:
Download .exe from Releases
Double click to play

🎮 How to Play
Click on cards to flip them
Match identical pairs
Complete the board with minimum moves
Try to beat your best score!
